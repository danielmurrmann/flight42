## Text on Background

### Primary

Use the primary colors for the most important actions and visuals you would like to emphasis to the user.

``primary-color`` color with text in ``text-on-primary-color``.
<div className="color-sample primary-color">On Primary</div>

### Secondary

Use the secondary colors for the secondary important actions and visuals you would like to emphasis to the user, but not as much as the primary things.

``secondary-color`` color with text in ``text-on-secondary-color``.
<div className="color-sample secondary-color">On Secondary</div>
